# Changelog

## SpNeigh 0.99.0

*Initial Bioconductor submission*

### New Features

- Introduced the
  [`RunSpatialDE()`](https://github.com/jinming-cheng/SpNeigh/reference/RunSpatialDE.md)
  function for spatial differential expression using spatial weights.
- Added
  [`ComputeSpatialEnrichmentIndex()`](https://github.com/jinming-cheng/SpNeigh/reference/ComputeSpatialEnrichmentIndex.md)
  for enrichment quantification.
- Implemented
  [`GetBoundary()`](https://github.com/jinming-cheng/SpNeigh/reference/GetBoundary.md),
  [`GetRingRegion()`](https://github.com/jinming-cheng/SpNeigh/reference/GetRingRegion.md),
  and
  [`RemoveOutliers()`](https://github.com/jinming-cheng/SpNeigh/reference/RemoveOutliers.md)
  for spatial region detection.
- Added
  [`PlotSpatialExpression()`](https://github.com/jinming-cheng/SpNeigh/reference/PlotSpatialExpression.md)
  and other visualization tools.
